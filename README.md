# BalmJS template - react-expert
> A React project with router

## Installation

```sh
npm install -g balm-cli
balm init react-expert my-project

cd my-project
yarn install
# or
npm install
```

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```

> thx [BalmJS](http://balmjs.com/)
